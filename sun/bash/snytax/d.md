## 条件运算符
```
1. 条件表达式要放在方括号之间，并且要有空格
```

## 注释
```
1. 每一行添加注释
2. 每一行加个#符号太费力了，可以把这一段要注释的代码用一对花括号括起来，定义成一个函数，没有地方调用这个函数，这块代码就不会执行，达到了和注释一样的效果。
```
## 数组
```
1. bash支持一维数组（不支持多维数组)
2. 使用()初始化数组,不要使用逗号，着会让解释器认为只有一个元素
3. 可以使用中括号单独初始化元素，可以不使用连续的下标，而且下标的范围没有限制。
4. ${array_name[index]} 读取数组
5. 获取所有的元素${array_name[*]}   ${array_name[@]}
6. 获取数组长度的方法与获取字符串长度的方法相同 使用# 运算符

```
## 字符串
```
1. 单引号 和 双引号 的区别
单引号里的任何字符都会原样输出，单引号字符串中的变量是无效的；
单引号字串中不能出现单引号（对单引号使用转义符后也不行）。

2. 拼接字符串
其实并没有所谓的拼接，只是把字符串放到一个""的下面greating 的双引号可要可不要
your_name="qinjx"
greeting="hello, "$your_name" !"
greeting_1="hello, ${your_name} !"
echo $greeting $greeting_1

3. 获取字符串长度
string="abcd"
echo ${#string} #输出 4

4. 提取子字符串
a="abcdefghijklmn"
b=${a:1:4}
echo $b

5. 查找子字符串
原理是第一个字符的位置，如果第一个字符不在，那么看后面的字符的情况
string="alibaba is a great company"
echo `expr index "$string" is`
```
